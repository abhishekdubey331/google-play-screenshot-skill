# Code of Conduct

## Our Standards

Contributors are expected to be respectful, constructive, and professional in all project interactions.

Examples of positive behavior:
- Giving and receiving feedback respectfully
- Focusing on actionable technical discussion
- Assuming good intent while reviewing contributions

Examples of unacceptable behavior:
- Harassment, discrimination, or personal attacks
- Trolling, insulting, or inflammatory comments
- Sharing private information without consent

## Enforcement

Project maintainers may remove comments, issues, or contributions that violate this code of conduct.

## Scope

This code applies in project spaces, including issues, pull requests, and discussions.
