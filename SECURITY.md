# Security Policy

## Reporting a Vulnerability

Please do not open public issues for suspected security vulnerabilities.

Report security concerns privately to the maintainers with:
- A clear description of the issue
- Reproduction steps
- Potential impact
- Any suggested mitigation

## Scope Notes

This repository includes prompt workflows and image-composition scripts.

Common security risks to avoid:
- Committing API keys or MCP credentials
- Committing proprietary app screenshots or user data
- Running untrusted prompts/scripts with production credentials
