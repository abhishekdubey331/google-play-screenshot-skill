# Security Policy

## Reporting a Vulnerability

Please do not open public issues for suspected security vulnerabilities.

Report security concerns privately using one of these channels:
- GitHub Private Vulnerability Reporting for this repository, if enabled
- Contact the repository owner privately through the contact options listed on their GitHub profile before disclosure

Include:
- A clear description of the issue
- Reproduction steps
- Potential impact
- Any suggested mitigation

## Scope Notes

This repository includes prompt workflows and image-composition scripts.

Common security risks to avoid:
- Committing API keys or MCP credentials
- Committing proprietary app screenshots or user data
- Running untrusted prompts/scripts with production credentials
